# bunq.js
This is an API client for the Bunq API.

## Installation
* Run `yarn install`
* Make sure you got an API key from Bunq

## Usage
* The Bunq client accepts some options

```
{
  apiKey: '',
  ipAddreses: [
  ]
}
```
